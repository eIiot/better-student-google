Better Student Search

Why?

I was tired of searching for chemistry help, but only getting results from sites like Chegg and Bartleby, where the answers are stuck behind a paywall. This site simply generates a Google search result with those sites emitted. To request a site be added to the filter, please submit an issue on GitHub.


Browser Extension
  • Download Tampermonkey, then
  • Install the UserScript

To use the extension, simply highlight any text, right click, select Tampermonkey and then Search with BHS.sh
